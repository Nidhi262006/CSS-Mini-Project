This mini project demonstrates a simple and clean navigation bar UI built using pure HTML and CSS. The goal was to strengthen core frontend concepts such as layout alignment, spacing, and positioning without relying on JavaScript or external frameworks.

Features:

Flexbox-based layout

Clean and structured UI

Proper spacing and alignment

Image fitting inside cards

Tech Stack:

HTML

CSS

Note:
Currently optimized for laptop and desktop screens. Responsiveness will be added in future updates.
